# Phantom — 200 Agents. 3 Humans.

> Ghost workforce platform — deploy 50-200 AI agents that handle your entire back office. You just steer.

![Screenshot placeholder](https://via.placeholder.com/1200x630/0a0a0f/7c3aed?text=Phantom)

Built for Claude Mythos.

## Stack

- Next.js 14 (App Router, static export)
- Three.js via @react-three/fiber + @react-three/drei
- Tailwind CSS
- Deploy: Netlify

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Outputs to `out/` — deploy to Netlify as a static site.

## Features

- Glowing agent grid 3D visualization with pulse waves
- Agent deployment configuration
- Task orchestration overview
- Human oversight dashboard
